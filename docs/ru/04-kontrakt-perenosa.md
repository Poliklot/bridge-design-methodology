# Контракт переноса

Макет, подготовленный по BRIDGE, раскрывает достаточно информации, чтобы реализацию не приходилось строить на догадках.

## Обязательные данные

Контракт должен содержать:

- корневые фреймы адаптивов с `[bp=...]`;
- стабильные идентификаторы слоёв;
- иерархию, Auto Layout, ограничения, позиционирование и обрезку из метаданных Figma;
- тип каждого элемента;
- геометрию и текстовые метрики для каждой контрольной ширины;
- содержимое и правила его сравнения между адаптивами;
- вложенность, порядок соседних элементов и видимость;
- известные `href` и `action`, а также оставшиеся черновые `[link]` и `[control]`;
- существующие цели ссылок, модальных окон, состояний и форм;
- правила экспорта сложных визуалов;
- высоту и ожидаемое поведение при переполнении;
- явно оформленные исключения.

Тип элемента определяется по метаданным Figma и BRIDGE-тегам. Например, Figma различает текст, изображение и экземпляр компонента, а BRIDGE добавляет страницу, секцию, действие, модальное окно, состояние, декор или экспортируемый ресурс.

## Пример модели данных

```json
{
  "methodology": "BRIDGE",
  "breakpoints": [1200, 320],
  "tree": {
    "1200": { "hero": ["hero-title", "contact-cta"] },
    "320": { "hero": ["hero-title", "contact-cta"] }
  },
  "elements": [
    {
      "id": "hero-title",
      "type": "text",
      "content": {
        "text": "Launch your store in one day",
        "sameAcrossBreakpoints": true
      },
      "layout": {
        "mode": "flow"
      },
      "fields": {
        "1200": { "left": 72, "top": 128, "width": 612, "fontSize": 58 },
        "320": { "left": 20, "top": 88, "width": 280, "fontSize": 34 }
      }
    },
    {
      "id": "contact-cta",
      "type": "button",
      "content": {
        "text": "Contact us"
      },
      "interaction": {
        "type": "modal",
        "targetKey": "contact-modal"
      }
    }
  ],
  "targets": [
    {
      "id": "contact-modal",
      "type": "modal"
    }
  ]
}
```

Конкретный адаптер может создать одну базовую структуру и затем детерминированно применить к ней значения для разных контрольных ширин.
