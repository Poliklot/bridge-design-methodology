# BRIDGE validator

This directory contains the first machine-readable seeds for future BRIDGE validation.

Current files:

- [`rules.json`](rules.json) — initial rule catalog for checklist and linter design.

Related documentation:

- [Validation and autochecks](../docs/11-validation-and-autochecks.md)
- [Hard cases and edge cases](../docs/10-hard-cases-and-edge-cases.md)
- [Project roadmap](../docs/12-project-roadmap.md)

The catalog is not an executable validator yet. It is a shared rule vocabulary for future CLI, Figma extraction, reports, and adapter certification.
